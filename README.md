# Simple Blog CLI Application (Node.js)

A simple command-line blog application built with Node.js that allows users to create, list, view, and delete blog posts. Posts are stored locally in a JSON file.

## Features

- Create a post
- List posts
- View a post
- Delete a post
- Store posts in a local JSON file

## How to run

```bash
node app.js
```
